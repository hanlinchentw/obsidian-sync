### Solution:
one by one, O(n^2)
```
class Solution {
public:
	string longestPalindrome(string s) {
		for (int i = 0; i < s.length(); i++) {
			int j = i < s.length() - 1 && s[i] == s[i+1] ? i+1 : i;
			while (i >= 0 && j < s.length() && s[i] == s[j]) {
				i--;
				j++;
			}
		}
	}
};
```
